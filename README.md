# Snooker
A school project about Snooker made using Java, specifically the JavaFX framework.

## Requirements
- Java 17 or higher
- download from [Azul](https://www.azul.com/downloads/?package=jdk#zulu) if needed.

## How to run
- Download the latest .jar from the [Release](../../releases) section.
- Run it using `java -jar Snooker-vX.X.X.jar` (replace X.X.X with the downloaded version)
